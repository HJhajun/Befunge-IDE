# Befunge-IDE

웹 기반 Befunge 전용 IDE입니다. (HTML/CSS/JavaScript)

## 실행
정적 파일 서버 또는 브라우저에서 `index.html`을 열어 사용합니다.

## 포함 기능
- 2차원 격자 편집(셀당 1문자, 행/열 삽입·삭제, 크기 변경)
- 사각형 선택/드래그 선택/Shift 확장
- 선택 영역 이동(Alt+방향키)
- 복사/잘라내기/붙여넣기/회전 붙여넣기/좌우·상하 반전
- Undo/Redo
- Befunge 이동 커서(방향키 기반) + 이동 기록 Backspace
- Befunge-93 실행기(실행/일시정지/Step/속도/리셋)
- 디버거(현재 위치, 이동 경로, 스택, 출력, 입력, 브레이크포인트)
- 시각화(방향/경로/문자열 모드/루프 히트맵)
- 프로젝트 기능(.bf 저장/열기, 최근 파일, 예제 코드)

## 단축키
- 화살표: Befunge 커서 이동(방향 반영)
- Shift+화살표: 선택 확장
- Alt+화살표: 선택 영역 이동
- Backspace: 현재 칸 삭제 + 이동 기록 기준 이전 위치로 복귀
- Ctrl/Cmd+Z: Undo
- Ctrl/Cmd+Y: Redo
- Ctrl/Cmd+C/X/V: 복사/잘라내기/붙여넣기

## 참고
- Befunge-98은 예정 상태이며 현재 엔진은 Befunge-93 기준으로 구현되어 있습니다.
